Представляем вашему вниманию веб - приложение, которое реализует логику запроса к Блокчейну Ethereum
и возвращает различную информацию относительно совершенного запроса.

возможности приложения:
Предоставляет состояние баланса конкретного токена или группы токенов ( пользователь сам передает ссылку на токен )
Предоставляет топ токенов на текущий период времени
Предоставляет топ токенов вместе с их транзакциями на текущий период времени
Предоставляет нам возможность получить более детализированную информацию о конкретном токене.


WEB APP - FAST API
размечены конкретные энд поинты для получения информации о токенах.
/get_balance/{adress} - позволяет нам получить информацию о балансе конкретного токена
/get_balance_batch/{batch} - позволяет нам получить информацию о балансе нескольких токенов
/get_top/{n} - позволяет нам получить информацию о топе токенов (n - список адресов)
/get_top_with_transactions/{n} - позволяет нам получить топ токенов с транзакциями(n - список адресов)
/get_token_info/{adress} - позволяет получать детализированную информаци о токене


Структура обращения

1. Полученный запрос от пользователя передается абстракции
1.1  абстракция обращается к конкретной реализации. в нашем случае БлокЧейн Эфириум с Полигон
1.2.1 конкретная абстракция в теле своей логики выполняет соответвующий код и получает данные от блокчейна
1.2.2 конкретная абстракция возвращает код абстракции.
1.1.1 Абстракция возвращает результат пользователю
